# Pt Unsqueeze
Adds a singleton dimension at the specified position in the input tensor.
For example, if a tensor has shape (3, 4) and `dim=1`, the output will have shape (3, 1, 4).

## Input
| Name | Data type |
|---|---|
| tens | Tensor |
| dim | Int |
