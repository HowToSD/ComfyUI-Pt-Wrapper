# Pt Show Text
Displays PyTorch tensor as a string. Note that the tensor is partially printed out when
the tensor contains more elements than the threshold set internally.

## Input
| Name | Data type |
|---|---|
| tens | Tensor |

## Output
| Data type |
|---|
| String |
