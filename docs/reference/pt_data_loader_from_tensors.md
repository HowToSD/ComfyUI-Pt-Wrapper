# Pt Data Loader From Tensors
Creates a Torchvision Dataloader from a pair of tensors.

Parameters  

**x**  
  Specify a tensor name that contains x.
  
**y**  
  Specify a tensor name that contains y.

## Input
| Name | Data type |
|---|---|
| x | Tensor |
| y | Tensor |
| batch_size | Int |
| shuffle | Boolean |
