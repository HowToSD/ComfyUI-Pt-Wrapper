# Pt Prod
Computes the product of a PyTorch tensor along the specified dimension(s).

Specify the dimension(s) in the `dim` field using an integer as shown below:
```
0
```
or
```
1
```

## Input
| Name | Data type |
|---|---|
| tens | Tensor |
| dim | String |
| keepdim | Boolean |

| Data type |
|---|
| Tensor |
